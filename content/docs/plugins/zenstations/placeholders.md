---
title: "Placeholders (PlaceholderAPI)"
description: "Variables disponibles para progreso y forjas."
---

Identificador: `zenstations`.

| Placeholder | Descripción |
| :-- | :-- |
| `%zenstations_slots_limit%` | Límite de slots activos. |
| `%zenstations_slots_active%` | Slots en uso. |
| `%zenstations_slots_available%` | Slots disponibles. |
| `%zenstations_unclaimed_count%` | Ítems finalizados sin reclamar. |
| `%zenstations_level%` | Nivel de herrería. |
| `%zenstations_xp%` | XP actual. |
| `%zenstations_xp_required%` | XP requerida para el siguiente nivel. |
| `%zenstations_xp_booster_multiplier%` | Multiplicador activo. |
| `%zenstations_xp_booster_remaining%` | Tiempo restante del booster. |
| `%zenstations_forge_remaining_<slot_id>%` | Tiempo restante o estado por slot. |
