---
title: "Comandos"
description: "Comandos de jugador y administración."
---

| Comando | Descripción | Permiso |
| :-- | :-- | :-- |
| `/rankup` | Subir al siguiente rango | — |
| `/ranks` | Abrir menú GUI de rangos | — |
| `/rankupmax` | Subir rangos hasta el máximo | — |
| `/prestige` | Subir al siguiente prestigio | — *(requiere prestigios activos)* |
| `/prestigemax` | Subir prestigios en bucle (máx 100 iteraciones) | — *(requiere prestigios activos)* |
| `/ranktop` | Abrir leaderboard de rangos | — |
| `/prestigetop` | Abrir leaderboard de prestigios | — *(requiere prestigios activos)* |
| `/zenrankups reload` | Recarga configs | `zenrankups.admin` |
| `/zenrankups setrank <player> <rank>` | Asigna rango | `zenrankups.admin` |
| `/zenrankups setprestige <player> <nivel>` | Asigna prestigio | `zenrankups.admin` |
| `/zenrankups reset <player>` | Resetea rango y prestigio | `zenrankups.admin` |
| `/zenrankups info <player>` | Muestra info del jugador | *Sin permiso* |
