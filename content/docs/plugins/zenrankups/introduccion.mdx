---
title: "Introducción a ZenRankups"
description: "Rankups y prestigios con requisitos flexibles, GUI y webhooks."
---

**ZenRankups** es un plugin avanzado de **rankups** y **prestigios** con soporte de base de datos SQLite/MySQL, requisitos flexibles (economía, puntos, placeholders), recompensas por comandos, menús GUI y webhooks de Discord.

## 🔌 Integraciones opcionales

- **Vault** (economía)
- **PlayerPoints**
- **Eco/EcoBits**
- **PlaceholderAPI**

Si una integración no está instalada, el plugin lo detecta y evita usarla, mostrando advertencias si un requisito la necesita.
