Comando principal: `/zenfairy` (aliases: `/zf`, `/souls`).

## 🎮 Jugador

- `/zf menu [tipo]` — Abre el menú del tipo de alma (default: `fairy`).

## 🛠️ Admin

- `/zf reload` — Recarga configuración y tipos de almas.
- `/zf set <tipo>` — Marca el bloque mirado como alma de ese tipo.
- `/zf remove` — Elimina el alma del bloque mirado.
- `/zf location [tipo]` — Menú de ubicaciones del tipo.
- `/zf reset <jugador> <tipo>` — Resetea datos del jugador.
