---
title: "Formato de tiempo"
description: "Sufijos aceptados para cooldowns y hitos."
---

ZenRewards usa un formato de tiempo simple y legible:

| Sufijo | Significado | Ejemplo |
| :-- | :-- | :-- |
| `s` | Segundos | `30s` = 30 segundos |
| `m` | Minutos | `15m` = 15 minutos |
| `h` | Horas | `1h` = 1 hora |
| `d` | Días | `7d` = 7 días |

Se puede usar en:
- Cooldowns de recompensas (`rewards.yml`)
- Hitos de playtime (`playtime.yml`)
- Cooldown de Daily Gifts (`daily-gifts.yml`)

Salida formateada: `2d 5h 30m 15s`
