## ✅ Pasos de instalación

1. **Descarga** el archivo `ZenRewards.jar`.
2. **Colócalo** en la carpeta `plugins/` de tu servidor.
3. **Inicia el servidor** para generar los archivos de configuración.
4. **Edita la configuración** en `plugins/ZenRewards/`.
5. **Aplica cambios** con `/zr reload` sin reiniciar.

---

## 🔌 Dependencias opcionales

| Plugin | Función |
| :-- | :-- |
| **PlaceholderAPI** | Habilita los placeholders `%zenrewards_...%` |

> Si PlaceholderAPI está instalado, ZenRewards lo detecta automáticamente.
